# Übung 18 - Mehrdimensionale Arrays
In diesem Projekt finden Sie vier Klasse: `Main`, `FileInAndOut`, `Spieler` und `SpielerVerwaltung`. Die `Main`-Klasse dient wie immer dazu, den Programmablauf zu bestimmen. Sie finden bereits ein einfaches Menü vor. Die `Spieler`-Klasse stellt einen Spieler dar, welcher Ihre Spiele spielen kann. Es wird eine eindeutiger Name und die Spielstatistik gespeichert. Die `Spielerverwaltung` dient dazu, um mehrere Spieler speichern zu können. Die `FileInAndOut`-Klasse schreibt die Daten aus der Spielerverwaltung in eine .txt-Datei. Bei neuerlichen Starten des Programms liest die Klasse die Textdatei ein und speichert die Werte wieder in der Spielerverwaltung. 

Ihre Aufgabe ist es nun das vorliegende Projekt zu erweitern:

## 1. Aufgabe

Programmieren Sie das Spiel "Tic Tac Toe". 

## 2. Aufgabe

Programmieren Sie das Spiel "Schiffe versenken".

## 3. Aufgabe

Programmieren Sie das Spiel "4 Gewinnt". 

**Gilt für alle Aufgaben:** Programmieren Sie fehlerfreie Spiele! Sie dürfen die bereits vorhandenen Klassen und Methoden verwenden, wenn Sie sie benötigen. Vor allem soll auch die Statistik für jedes Spiel funktionieren.
